# Firmware Upgrade

## Prepare

Connect the device to the PC via a USB cable. Power on the device with 12/24 VDC (recommended).

## Steps

- Download the latest firmware upgrade package.
  https://github.com/leslieyang-amec/N323/releases
- Decompress the package.
- Execute the "FirmwareUpgade.bat" file. ( [DECOMPRESS_PATH]\xAisUtility\FirmwareUpgrade.bat )
- Select the related serial port, then press the "Connect" button.
- Browse the firmware upgrade file. The default path is [DECOMPRESS_PATH]\FirmwareUpgrade_BinFile\AisAtoN\*.pdfu
- Press the "Start" button.
